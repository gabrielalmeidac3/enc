Política de Privacidade Este aplicativo ("Subidor de conteúdo") é utilizado exclusivamente para publicação automática de conteúdo em páginas do Facebook de propriedade do próprio usuário. Não coletamos, armazenamos ou compartilhamos dados pessoais de usuários finais. Os dados acessados pela API do Facebook são usados apenas para: - Publicar vídeos e Reels em páginas autorizadas - Gerenciar conteúdo da própria página Nenhuma informação é vendida, transferida ou utilizada para fins comerciais.
